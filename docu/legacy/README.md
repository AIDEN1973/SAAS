## 🔄 전환 이유

### Intent 기반의 한계

1. **확장성 부족**: 147개 Intent 관리 부담
2. **복잡한 대화 처리 불가**: 단일 Intent만 처리 가능
3. **유지보수 비용 높음**: ~10,000줄 코드
4. **자연스러운 대화 어려움**: 명령어 파싱 방식

### Agent 기반의 장점

1. **확장성 향상**: 15개 Tool로 모든 기능 커버
2. **복잡한 대화 지원**: 멀티턴, 멀티 액션 처리
3. **유지보수 비용 감소**: ~500줄 코드 (95% 감소)
4. **자연스러운 대화**: LLM이 대화 주도
